To run a live demo of the TableNG library, follow these steps:

Step 1: Build the library (already done)
npm run build
Step 2: Create a new Angular application
ng new tableng-demo --routing --style=css
Step 3: Install the built library
cd tableng-demo
npm install ../dist/tableng
Step 4: Import and use the TableNG module
// app.component.ts
import { Component } from '@angular/core';
import { TablengModule, TableConfig } from 'tableng';

@Component({
  selector: 'app-root',
  imports: [TablengModule],
  template: `
    <tng-table
      [config]="tableConfig"
      [data]="tableData">
    </tng-table>
  `
})
export class AppComponent { ... }

üéØ Key Features

  • Virtual Scrolling: Handle large datasets (10k+ rows) efficiently
  • Sorting & Filtering: Built-in sorting and filtering capabilities
  • Column Resizing: Drag column borders to resize
  • Column Reordering: Drag and drop columns to reorder
  • Inline Editing: Edit cells directly in the table
  • Row Selection: Single or multiple row selection
  • Sticky Headers: Headers stay visible while scrolling
  • Theming: Customizable themes and styling
  • State Persistence: Save and restore table state
  • Accessibility: Full keyboard navigation and screen reader support
